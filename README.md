# 旅游信息管理系统 java1016

## 项目概述

本项目是基于 Spring Boot + Vue 的前后端分离的旅游信息管理系统，旨在提供一套完善的旅游信息管理解决方案。

## 技术栈

- 前端：Vue
- 后端：Spring Boot
- 数据库：MySQL
- 构建工具：Maven、Node.js

## 运行环境

- 开发工具：IDEA
- JDK版本：1.8
- MySQL版本：5.7
- Maven版本：3.6.1
- Node.js版本：12.22.12（其他版本也可自测）

## 功能模块

### 前台

- 展示旅游新闻
- 景区信息展示
- 美食信息展示
- 旅游路线介绍
- 在线留言功能

### 后台管理员

- 账号管理
- 地区管理
- 景点信息管理
- 地方美食管理
- 旅游路线管理
- 订单信息管理
- 新闻管理
- 系统管理

### 普通用户

- 旅游路线查询
- 订单信息管理

## 系统角色

- 系统管理员
- 普通用户
